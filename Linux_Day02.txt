File Permissions in Linux
rw-r--r-- Dockerfile

In Linux, file permissions are categorized into 3 ways;
	1. User permissions (u)
	

	2. Group permissions (g)


	3. Others permissions (o)

For every category, we will have 3 permissions
	read (r)
	write (w)
	execute (x)

chmod u-r Dockerfile

o - no permissions
1 - execute
2 - write
3 - 2+1 - write + execute
4 - read
5 - 4+1 - read + execute
6 - 4 + 2 - read + write
7 - 6+ 1 - read + write + execute
